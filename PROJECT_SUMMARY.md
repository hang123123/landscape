## 🚀 启动指南

### 快速启动
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动应用
python main.py

# 3. 访问应用
open http://localhost:8000
```

### 完整配置
```bash
# 1. 复制环境变量文件
cp env.example .env

# 2. 编辑.env文件，配置阿里百炼API密钥（可选）
# ALIBABA_DASHSCOPE_API_KEY=your_api_key

# 3. 使用智能启动脚本
python start.py
```
